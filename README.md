# MentoAI

A small fullstack GenAI project containing backend and frontend.

Structure
- backend/: Node/Express backend
- frontend/: Vite + React frontend

Quick start

Backend:
```bash
cd backend
npm install
npm start
```

Frontend:
```bash
cd frontend
npm install
npm run dev
```

Notes
- Add a `.env` in `backend/` for secrets.
- If you want `chatgpt/` handled specially, tell me whether to add it as a submodule, embed it, or ignore it.
